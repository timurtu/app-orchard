# App Orchard

An app where users suggest app ideas.

## Building from source

### Install
- `git clone https://github.com/timurtu/app-orchard`
- `cd app-orchard`
- `npm install`

### Running and Testing
- `npm start`

### Production
- go to __http://localhost:3000__

### Hot loading
- go to __http://localhost:8080__
